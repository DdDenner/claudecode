# Story 3.3 - Implementar Formulário de Contato

**ID:** 3.3  
**Agente:** @dev  
**Tamanho:** Small  
**Tempo Estimado:** 1 dia

## Descrição
Criar formulário de contato funcional integrado com email. Campos: Nome, Email, Tipo de Projeto, Mensagem. UX clara e intuitiva.

## Critério de Aceitação
- [ ] Formulário validado (email, campos obrigatórios)
- [ ] Email recebido em inbox (usar Resend ou FormSpree)
- [ ] Mensagem de sucesso após envio
- [ ] UX clara com estilos Tailwind
- [ ] Mobile-friendly

## Notas
- Opção recomendada: Resend (SDK do Next.js) ou FormSpree (sem backend)
- Alternativa simples: WhatsApp link direto para contato
- Spam protection: Honeypot ou reCAPTCHA v3
