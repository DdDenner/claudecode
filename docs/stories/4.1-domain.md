# Story 4.1 - Configurar Domínio & Hospedagem

**ID:** 4.1  
**Agente:** @devops  
**Tamanho:** Small  
**Tempo Estimado:** 1 dia

## Descrição
Comprar domínio e configurar hospedagem em Vercel. DNS, SSL/HTTPS, e opcionalmente email de domínio.

## Critério de Aceitação
- [ ] Domínio escolhido (ex: denner.design, dennerdesign.com.br)
- [ ] Domínio registrado (Namecheap, Google Domains, etc)
- [ ] Vercel configurado e conectado a repositório GitHub
- [ ] DNS apontando para Vercel
- [ ] SSL/HTTPS ativo e funcionando
- [ ] Domain configurado em Vercel (Environment variables se necessário)

## Notas
- Recomendação: denner.design ou dennerdesign.com.br
- Hospedagem: Vercel (automático com Git + Next.js)
- Email: Opcional, pode ser feito depois com Namecheap Forwarding
