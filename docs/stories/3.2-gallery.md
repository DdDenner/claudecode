# Story 3.2 - Implementar Galeria de Projetos

**ID:** 3.2  
**Agente:** @dev  
**Tamanho:** Medium  
**Tempo Estimado:** 2 dias

## Descrição
Implementar galeria interativa com cards dos 10-15 projetos. Grid responsivo, imagens otimizadas (WebP + fallback), hover effects, e opcionalmente modal ao clicar.

## Critério de Aceitação
- [ ] Todos os 10-15 projetos carregando
- [ ] Grid responsivo (2-3-4 colunas conforme tela)
- [ ] Imagens otimizadas (max 100KB cada)
- [ ] Hover effects profissionais (scale, overlay, etc)
- [ ] Modal/lightbox funcional (opcional mas desejado)

## Notas
- Input: `docs/portfolio-cards.md` (Story 1.3)
- Ferramentas: Next.js Image component para otimização, Framer Motion para hover
- Performance: Lazy loading em mobile
