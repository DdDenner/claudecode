# 🎨 Guia Rápido: Como Selecionar seus 10-15 Melhores Projetos

---

## 🚀 Processo em 4 Passos

### PASSO 1: Abra as Duas Fontes de Projetos

**Opção A - Google Drive (37 projetos)**
- Acesse seu Google Drive
- Procure pela pasta com seus projetos
- Revise cada um

**Opção B - Pasta Local (52+ arquivos)**
```
C:\Users\denne\OneDrive\Área de Trabalho\Artes
```
- Abra o Explorer
- Veja os arquivos CorelDRAW, PSD, etc

---

### PASSO 2: Critério de Seleção (Faça 3 Perguntas)

Para CADA projeto, responda:

**1. "Este projeto fica BONITO na tela?"**
- ✅ Sim → Continua analisando
- ❌ Não → Pula para o próximo

**2. "Representa bem meu trabalho?"**
- ✅ Sim, eu tenho orgulho deste → Continua
- ❌ Não, tenho outros melhores → Pula

**3. "Ajuda a DIVERSIFICAR o portfólio?"**
- ✅ Sim, tem categoria diferente → Adiciona
- ❌ Não, é parecido com outro que já escolhi → Talvez pule

---

### PASSO 3: Escolha entre 10-15

**Regra de Ouro:**
- Mínimo: 10 (quantidade suficiente para impressionar)
- Máximo: 15 (não fica monótono)
- Meta: 12-13 (número perfeito)

**Distribuição sugerida:**
```
Identidade Visual:  3-4 projetos  (logo, brand)
Embalagem:          2-3 projetos  (rótulo, caixa)
Sinalização:        2-3 projetos  (placa, outdoor)
Flyers:             2-3 projetos  (material impresso)
Outros:             1-2 projetos  (digital, arte, etc)
──────────────────────────────────
TOTAL:              12-15 projetos
```

---

### PASSO 4: Documente cada um

Para cada projeto selecionado, anote:

```
Nome:        [Ex: Identidade Visual - Padaria Local]
Categoria:   [Ex: Identidade Visual]
Cliente:     [Ex: "Padaria do João" ou "Personal Project"]
Descrição:   [Ex: "Logo, cartão, papel timbrado e adesivo"]
Imagem:      [Ex: "Drive/pasta-nome/arquivo.jpg"]
```

---

## 💡 Dicas Práticas

### ✅ Escolha ESTE tipo de projeto
- Trabalho que resultou em cliente feliz
- Projeto com muita criatividade/inovação
- Caso de sucesso (resultado mensurável)
- Visual impactante e profissional
- Tecnicamente bem executado

### ❌ Evite ESTE tipo
- Trabalhos muito antigos/datados
- Projetos que não refletem seu nível atual
- Cópias/variações do mesmo tema
- Trabalhos incompletos
- Design muito experimental (a menos que seja seu diferencial)
